## ✨ Features

### Core Features
- **🤖 Interactive AI Chatbot** - Ask questions about my skills, experience, and background. The chatbot is programmed with comprehensive information about my professional journey and can answer various questions in real-time.
- **🌓 Dark/Light Mode Toggle** - Seamless theme switching with persistence using localStorage. Users can switch between light and dark modes, and their preference is saved for future visits.
- **📄 CV Upload & Download** - Easy resume management for employers. I can upload my latest CV, and employers can download it directly from the portfolio. The upload area supports drag-and-drop functionality.
- **🎨 Animated UI Elements** - Glitch text effects on the name header, floating skill cards, progress bars with pulse animations, bounce effects on upload area, and shine effects on project cards.
- **📱 Fully Responsive** - Perfect viewing on all devices from mobile phones to desktop monitors. The layout adapts seamlessly using CSS Grid and Flexbox.
- **⚡ Smooth Scroll Navigation** - All anchor links provide smooth scrolling to different sections of the portfolio.
- **💾 Local Storage Integration** - Theme preferences are saved and loaded automatically.

### Visual Highlights
- Purple-themed color scheme with carefully chosen shades (not too dark!)
- Multiple CSS animations including float, bounce, pulse, and shine effects
- Parallax background effects that respond to mouse movement
- Typewriter text effect on the subtitle when the page loads
- Hover transformations on all interactive elements with scale and translate effects
- Gradient animations in the background for visual interest
- Glitch effect on the main name for a tech-forward look
